# Milestones

This folder turns the product roadmap into actionable release plans.

## How to use this folder

1. Pick the active milestone.
2. Move one task at a time from unchecked to in progress to complete.
3. Keep implementation notes close to the milestone.
4. Update `PROJECT_STATUS.md` after meaningful progress.
5. Add release notes before merging a milestone.
